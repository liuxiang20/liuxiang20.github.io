# C++ _ 面向对象 _ 封装



### 1、含义：

将生活中事物的属性和行为加以不同的权限集合成一个整体，这个这个整体就是类，这项技术就是封装。封装是C++面向对象三大特性之一。

### 2、语法：

**语法：** `class 类名{   访问权限： 属性  / 行为  };`

**Eg：**创建学生类，在public权限下创建成员属性为学生ID，姓名和年龄。创建构造函数对成员属性进行赋值操作。在main函数中实例化对象s,调用对象s里的打印成员函数打印相关数据。

```C++
class Student
{
public:
//构造函数，对成员属性进行赋值操作
    Student(int id,string name,string age)
    {
        m_ID = id;
        m_Name = name;
        m_Age = age;
    }
//成员函数打印操作，成员行为
    void print()
    {
        cout<< "学生ID："  << m_ID 
            << "学生姓名：" << m_Name
            << "学生年龄：" << m_Age
            << endl;
    }
//成员属性
    int m_ID;
    string m_Name;
    int m_age;
}

int main()
{
//对学生类实例化对象s
    Student s;
//调用成员函数并对成员属性赋值打印操作
    s.print(221,"刘祥"，"工程管理");
    
    system("pause");
    return 0;
}
```

### 3、权限

类在设计时，可以把属性和行为放在不同的权限下，加以控制。访问权限有三种：

1. public         公共权限  
2. protected      保护权限
3. private        私有权限

**Eg：**同样的引用上面案例，把不同成员属性赋予不同的权限，类外公有权限可以访问，保护权限和私有权限无法访问。

```C++
class Student
{
public:
//构造函数，对成员属性进行赋值操作
    Student(int id,string name,string age)
    {
        m_ID = id;
        m_Name = name;
        m_Age = age;
    }
//类内可以访问三种权限，类外只可以访问公有权限
    void print()
    {
        cout<< "学生ID："  << m_ID 
            << "学生姓名：" << m_Name
            << "学生年龄：" << m_Age
            << endl;
    }
//成员属性
public:
string m_Name;
protected:
    int m_ID; 
private:
    int m_age;
}

//类外调用不同的成员函数
//解决方法可以通过友元方式在类外访问，在后续单独讲解
void test01（）
{
    //cout<< "学生ID："  << m_ID << endl;      //类外无法访问
    cout<< "学生姓名："  << m_Name << endl;  
    //cout<< "学生年龄："  << m_Age << endl;   //类外无法访问
}

int main()
{
//对学生类实例化对象s
    Student s;
//调用成员函数并对成员属性赋值打印操作，这里是通过对象调用，可以给不同权限的成员函数看赋值。
    s.print(21,"张三"，"工程管理");
// 调用测试函数    
    test01();

    system("pause");
    return 0;
}
```
